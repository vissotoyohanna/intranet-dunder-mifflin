# 🏢 Dunder Mifflin Intranet - Scranton Branch

> **Status do Projeto:** 🚧 Em desenvolvimento

Este é um projeto Full Stack desenvolvido do zero para simular a Intranet corporativa da filial de Scranton da Dunder Mifflin (The Office). O objetivo é aplicar conceitos avançados de Front-End e Back-End, incluindo Controle de Acesso Baseado em Cargos (RBAC), design modular e gerenciamento de banco de dados.

## 🎯 Objetivos Técnicos
- Construir uma interface corporativa limpa, responsiva e acessível.
- Implementar arquitetura de arquivos CSS modular.
- Desenvolver uma API Back-End para gerenciamento de estoque e usuários.
- Estruturar um banco de dados relacional com diferentes níveis de permissão (Gerência, RH, Vendas, etc.).

## 🛠️ Tecnologias Utilizadas

**Front-End:**
- HTML5 (Semântico)
- CSS3 (Grid, Flexbox, Variáveis, Arquitetura Modular)

**Back-End & Infraestrutura (Em desenvolvimento):**
- Python
- Flask / FastAPI
- SQLite
- Autenticação e Criptografia de Senhas

## 🚀 Funcionalidades Atuais
- [x] **Arquitetura Front-End:** Divisão de arquivos CSS (Global, Dashboard, Diretório e Login).
- [x] **Tela de Login Restrito:** Interface corporativa de autenticação.
- [x] **Dashboard Inicial:** Mural de comunicados oficiais e painel lateral de atalhos.
- [x] **Diretório Corporativo:** CSS Grid responsivo com o contato e ramal de todos os funcionários organizados por hierarquia.
